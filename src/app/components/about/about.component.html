This is blume.

This website shows the details of all the upcoming, trending, top-rated movies so that you can find them in one place i.e. here!

Want to make any suggestion, feedback?

Browse through thousands of catalogues of movies. Search for a particular movie or click on any movie card to get the detailed description about that movie, watch trailers, movie screenshots, movie casts photos and more!

This web app is powered by API's provided by TMDB. If you want to contribute to this project head over here! Any idea or contribution would be highly appreciated. I will be adding more features in the near future so stay tuned.
